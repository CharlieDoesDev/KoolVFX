This folder stores textures for the box environment. Place your base color (albedo) and normal map images for each face here.

Recommended naming convention:
- box_front_base.jpg / box_front_normal.jpg
- box_back_base.jpg / box_back_normal.jpg
- box_left_base.jpg / box_left_normal.jpg
- box_right_base.jpg / box_right_normal.jpg
- box_top_base.jpg / box_top_normal.jpg
- box_bottom_base.jpg / box_bottom_normal.jpg
